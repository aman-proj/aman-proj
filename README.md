## 🧠 How I Think

```
Problem arrives
      │
      ▼
 ┌──────────────────────────────────────┐
 │  1. CONSTRAINTS  →  What breaks?     │  ← Always here first
 │  2. ABSTRACTION  →  What's true?     │  ← Model before code
 │  3. PATTERNS     →  Seen this?       │  ← 500+ problems talking
 │  4. EXECUTION    →  Now build.       │  ← Clean, then fast
 └──────────────────────────────────────┘

I don't brute-force problems.
I don't brute-force systems.
Both deserve a mental model first.
```
